Multi-page Coffee Shop E-Commerce Site

A multi-page coffee shop website featuring two modal windows, two forms with client side validation, and a fully (frontend) functioning e-commerce page.

Technologies Used and Skills Demonstrated
  1. Html 5 markup
  2. SCSS, packaged to CSS, run through autoprefixer, and minified using Gulp
  3. All behavior coded with vanilla JavaScript.
  4. CSS Grid
  5. CSS Flexbox
  6. Two different modal windows, including the shopping cart and a café menu
  7. Mapbox API integration and display.

A fun Frontend Mentor project incorporating the EU's Countries REST API. There were a lot of fun challenges in this one, including a dark mode toggle. I built this with React, using Router and Hooks.

  1. JSX markup
  2. React.js, employing Router and Hooks
  3. REST API integration and data handling.
  4. CSS Grid
  5. CSS Flexbox
  6. CSS Variables, adjusted via JavaScript on dark mode toggle
